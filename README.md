# Modabber (مدبر) - Financial Dashboard

**Modabber** is a professional Arabic dashboard designed to help users track expenses and manage budgets effectively.

##  Technologies Used
* **HTML5** (Semantic Structure)
* **CSS3** (Advanced CSS Grid Layout)
* **Responsive Design** (RTL Support)

##  Key Features
* Custom-built dashboard using **CSS Grid** without external frameworks.
* Full support for Arabic language (Right-to-Left layout).
* Clean and organized UI for financial data visualization.
